# 用户操作手册

## 1. 登录与授权
访问系统首页，点击“Login with PingCode”按钮，跳转至 PingCode 授权页面。确认授权后，系统将自动跳转回控制台。

## 2. 数据同步
首次使用建议点击右上角的 **Sync Data** 按钮。系统将拉取您在 PingCode 中的所有项目及工作项，建立本地向量索引。这是智能匹配的基础。

## 3. 需求分析
1. 在首页拖拽上传需求文档（支持 Word, Markdown, TXT）。
2. 等待系统智能分析，结果将显示在下方列表中。
3. 列表包含：标题、优先级、预估工时。

## 4. 项目匹配与查重
1. 系统会自动根据需求内容推荐最匹配的项目。
2. 您也可以在“Target Project”下拉框中手动切换项目。
3. 选择项目后，系统会自动检测每一条需求是否与该项目中现有的工作项重复。
   - **New**: 全新需求。
   - **Similar**: 存在相似工作项（显示相似项标题）。

## 5. 导入 PingCode
确认需求列表无误后，点击 **Import to PingCode** 按钮。系统将批量创建工作项，并在完成后提示成功数量。
